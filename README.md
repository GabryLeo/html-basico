# 🌐 HTML Básico

Um projeto simples de site institucional desenvolvido com HTML puro, como parte dos meus estudos iniciais em desenvolvimento web.

## 📄 Páginas

- **index.html** — Página inicial
- **quem-somos.html** — Página sobre a empresa/equipe
- **contato.html** — Formulário de contato

## 🚀 Como visualizar

1. Clone o repositório:
```bash
git clone https://github.com/GabryLeo/html-basico.git
```
2. Abra o arquivo `index.html` no seu navegador

Não é necessário instalar nada!

## 🛠️ Tecnologias

- HTML5

## 📚 Sobre o projeto

Este foi um dos meus primeiros projetos em HTML, desenvolvido para aprender a estrutura básica de páginas web, uso de tags semânticas, links entre páginas e formulários.
